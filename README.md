# Just Breathe - Meditation App

A beautiful, warm-toned meditation timer app with calendar tracking.

## Features

- Multiple timer durations (1m, 3m, 5m, 10m, 15m, 20m, 25m, 30m)
- Beautiful circular progress indicator
- Post-meditation reflection journal
- Calendar view showing meditation days
- Timeline of all meditation sessions
- Gentle bell sound when timer completes
- Warm, calming color theme
- Fully responsive for mobile devices

## Deployment Options

### Option 1: GitHub Pages (Recommended - Free)

1. Create a GitHub account if you don't have one
2. Create a new repository (e.g., "meditation-app")
3. Upload `index.html` to the repository
4. Go to Settings > Pages
5. Select "main" branch and "/ (root)" folder
6. Your app will be available at: `https://yourusername.github.io/meditation-app/`

### Option 2: Netlify Drop (Easiest - No account needed)

1. Go to https://app.netlify.com/drop
2. Drag and drop the `index.html` file
3. Get an instant URL like: `https://random-name-123.netlify.app`
4. Bookmark it on your phone!

### Option 3: Vercel (Free)

1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in this directory
3. Follow the prompts
4. Get a URL like: `https://meditation-app.vercel.app`

### Option 4: Local Network (For testing)

1. Install a simple HTTP server: `python3 -m http.server 8000`
2. Find your computer's IP address
3. On your phone, go to: `http://YOUR_IP:8000`
4. Make sure phone and computer are on same WiFi

## Mobile Access

Once deployed, you can:
- Bookmark the URL on your phone's browser
- Add to home screen (iOS: Share > Add to Home Screen, Android: Menu > Add to Home Screen)
- Use it like a native app!

## Data Storage

All meditation sessions are stored locally in your browser's localStorage, so your data stays private and on your device.
